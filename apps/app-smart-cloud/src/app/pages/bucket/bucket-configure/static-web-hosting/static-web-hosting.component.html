Thiết lập Versioning
Cấu hình Bucket hoạt động như một WebServer dành cho Website tĩnh

Trên bucket chỉ chứa các dữ liệu tĩnh của website: HTML, CSS, JS, ảnh...
Bật Tắt
File chứa nội dung hiển thị trang chủ của website có link là Endpoint bên dưới
Ví dụ: index.html
File chứa nội dung hiển thị lỗi của website
Ví dụ: error.html
Lưu ý: Để website hoạt động hãy đảm bảo file đã được public
